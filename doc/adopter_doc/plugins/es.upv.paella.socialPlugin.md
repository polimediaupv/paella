# es.upv.paella.socialPlugin

This plugin show a pop-up for share the video content in different social networks.

![](images/socialPlugin.jpg)

## Plugin Type:
- [paella.ButtonPlugin](../developer/plugin_types.md)

## Config Example:

```json
{
	"es.upv.paella.socialPlugin": 
	{
		"enabled": true
	},
}
```
