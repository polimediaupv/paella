# es.upv.paella.liveStramingIndicator

This plugin adds an icon into the player for indicate to the user that the video is a live streaming.


## Plugin Type:
- [paella.VideoOverlayButtonPlugin](../developer/plugin_types.md)

## Config Example:

```json
{
	"es.upv.paella.liveStramingIndicator": 
	{
		"enabled": true
	},
}
```
