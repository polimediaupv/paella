# es.upv.paella.playButtonOnScreenPlugin

This plugin show a play button on the screen to start the video


## Plugin Type:
- [paella.EventDrivenPlugin](../developer/plugin_types.md)


## Config Example:

```json
{
  "es.upv.paella.playButtonOnScreenPlugin": {
    "enabled": true
  }
}
```
